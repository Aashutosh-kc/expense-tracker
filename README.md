# Expense Tracker

A simple expense tracking app built with **React + Vite** as part of my frontend learning journey.

## Live Demo:
https://expense-tracker-seven-kappa-49.vercel.app/
## Features

- Add expenses with a name and amount
- View a live list of all expenses
- See total spending calculated in real time
- Delete individual expenses

## What I Practiced

- React components and JSX
- `useState` for managing state
- Props and lifting state up
- Controlled inputs
- `.map()` for rendering lists
- `.reduce()` for calculating totals
